# 财联社电报监控

财联社电报实时抓取 + Claude AI 利好/利空分析 + 自选股实时报价，基于 PyQt6 的 macOS 桌面应用。

## 功能

- 定时抓取 [财联社电报](https://www.cls.cn/telegraph)，支持自定义间隔
- 调用本地 Claude CLI 分析每条新闻对 A 股的影响（利好↑ / 利空↓）
- 结果保存到 Excel，支持增量更新去重
- 顶部实时报价栏：输入股票代码，实时显示价格和涨跌幅
- 暗色主题 UI，运行日志颜色区分，数据表支持双击复制

## 安装

```bash
pip install -r requirements.txt
```

需要本地安装 [Claude Code CLI](https://claude.ai/code) 并完成登录。

## 运行

```bash
python3 cls_app.py
```

## 使用说明

| 功能 | 操作 |
|------|------|
| 开始定时监控 | 左侧设置参数 → 点「▶ 开始监控」 |
| 单次执行 | 点「⚡ 立即执行一次」 |
| 添加自选股报价 | 顶部输入6位股票代码 → 回车 |
| 复制股票代码 | 双击数据表中的股票代码格 |
| 打开 Excel | 点「📂 打开 Excel」 |

## 配置

首次运行自动生成配置文件 `~/.cls_monitor_config.json`，关闭应用时自动保存。

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 监控频率 | 每隔多少分钟抓取一次 | 5 分钟 |
| 加载次数 | 页面向下滚动加载的次数，越大抓取越多 | 3 次 |
| 等待超时 | 页面加载超时时间 | 20 秒 |
| Claude 路径 | claude 可执行文件路径，自动检测 | 自动 |
